# Resto-MAPS
Application mobile with React Native and Firebase
Finished application : https://www.youtube.com/watch?v=rkRU2NQiDN4

- Firebase authentication
- Firestore cloud (database)
- Favorites system
- Geolocation
- Ranking and Reviews (Restaurants)
